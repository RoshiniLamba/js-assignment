String Methods

String length()

The length property returns the length of the string(Including space)


String indexOf()

We make use of indexOf to find the position of a string from the given text


lastIndexOf()

lastIndexOf is used to find the last occurrence (location/position) of the string/character


string search()

The search() method searches a string for a specified value and returns the position of the match


Slice()

slice() extracts a part of a string and returns the extracted part in a new string.
The method takes 2 parameters: the start position, and the end position (end not included).


substring()

The substring() method returns the part of the string between the start and end indexes, or to the end of the string.


subStr()

The substr() method returns a portion of the string, starting at the specified index and extending for a given number of characters afterwards.


replace()

The replace() method replaces a specified value with another value in a string

syntax : const newStr = str.replace(regexp|substr, newSubstr|function)


concat()

Joins 2 or more string


trim()

The trim() method removes whitespace from both sides of a string


charAt()

The charAt() method returns the character at a specified index (position) in a string


charCodeAt()

The charCodeAt() method returns the unicode of the character at a specified index in a string
The method returns a UTF-16 code (an integer between 0 and 65535)


split()

A string can be converted to an array with the split() method
